# Visoth Portfolio

Personal portfolio site for Heng Ratanak Visoth, built with Vue 3 and Vite

## Tech Stacks

- Vue 3, Vite, TypeScript
- Tailwind CSS, Flowbite
- Animation: GSAP, AOS, Motion One
- UI and icons: Swiper, Font Awesome

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- pnpm

### Install

```bash
pnpm install
```

### Run the dev server

```bash
pnpm run dev
```

### Build for production

```bash
pnpm run build
```

### Preview the production build

```bash
pnpm run preview
```

## Scripts

- `pnpm run dev` - start Vite dev server
- `pnpm run build` - build for production
- `pnpm run preview` - preview the production build
- `pnpm run test:unit` - run unit tests with Vitest
- `pnpm run type-check` - run Vue + TypeScript type checks
- `pnpm run lint` - run ESLint (auto-fix enabled)
- `pnpm run format` - format `src/` with Prettier

## Project Structure

- `index.html` - app entry HTML
- `src/` - Vue components, routes, and app logic
- `public/` - static assets
